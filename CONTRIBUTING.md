# Contributing to AI Trader

Thank you for your interest in contributing to AI Trader! We believe in the power of open source to build better financial tools for everyone.

## Code of Conduct

This project connects people from around the world. Please be respectful, inclusive, and professional in all interactions. 

## How to Contribute

1.  **Fork the Repository**: Create your own copy of the project.
2.  **Clone the Repository**: `git clone https://github.com/YOUR-USERNAME/ai-trader_ts.git`
3.  **Create a Branch**: `git checkout -b feature/amazing-feature`
4.  **Make Changes**: Write your code and tests.
5.  **Commit Changes**: Use descriptive commit messages.
    -   `feat: add new strategy indicator`
    -   `fix: resolve race condition in risk check`
    -   `docs: update installation guide`
6.  **Push to Branch**: `git push origin feature/amazing-feature`
7.  **Open a Pull Request**: Submit your PR to the `main` branch.

## Development Guidelines

-   **TypeScript**: We use TypeScript for type safety. Please ensure no `any` types are used unless absolutely necessary.
-   **Linting**: Run `npm run lint` before committing to ensure code style consistency.
-   **Tests**: Add unit tests for new logic where appropriate.

## Reporting Issues

If you find a bug or have a suggestion, please open an issue in the GitHub repository. Provide as much detail as possible, including steps to reproduce the issue.

Thank you for helping make AI Trader better!
